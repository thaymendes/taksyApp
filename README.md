# iOS: Construindo uma lista de tarefas com MVC e View Code

Este repositório contém o código-fonte do projeto desenvolvido no curso **"iOS: Construindo uma lista de tarefas com MVC e View Code"** da Alura.

## Objetivo
O objetivo deste curso é ensinar os conceitos básicos do MVC (Model-View-Controller) e como implementá-lo em um aplicativo iOS utilizando View Code. O projeto final será um aplicativo simples de gerenciamento de tarefas, com as seguintes funcionalidades:

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Buscar tarefas

## Pré-requisitos
- Conhecimento básico de Swift
- Familiaridade com o Xcode
- Experiência com desenvolvimento de aplicativos iOS (opcional)

## Conteúdo

### Introdução ao MVC
- Conceitos básicos do MVC
- Arquitetura MVC em iOS
- Vantagens do MVC

### Criando a interface do usuário
- Utilizando Storyboards e View Code
- Criando layouts com Auto Layout
- Implementando tabelas (`UITableView`)

### Implementando a lógica do aplicativo
- Criando modelos de dados (Model)
- Controlando a lógica da aplicação (Controller)
- Interagindo com a interface do usuário (View)

### Gerenciando dados
- Armazenando dados localmente (UserDefaults)
- Trabalhando com arrays e dicionários

### Testando o aplicativo
- Testes unitários
- Testes de interface do usuário

## Como usar este repositório
1. Clone este repositório para sua máquina local.
2. Abra o projeto no Xcode.
3. Compile e execute o aplicativo no simulador ou em um dispositivo iOS.
